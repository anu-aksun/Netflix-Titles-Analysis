# Netflix Titles Analysis – Exploratory Data Analysis (EDA)

## Project Overview
Netflix has grown into one of the world’s largest content platforms, producing and hosting thousands of movies and TV shows across different countries and genres. Understanding the composition of this vast catalog is essential for improving content strategy, regional reach, and audience engagement.

This project explores the Netflix Titles Dataset to uncover trends, patterns, and insights related to content production, genre popularity, and regional diversity.

---

## Objective
- Analyze Netflix’s content catalog using exploratory data analysis  
- Identify trends in content production over time  
- Understand genre popularity and content distribution  
- Explore regional diversity in Netflix’s offerings  
- Derive actionable insights to improve content strategy and audience engagement  

---

## Dataset
- Dataset: https://www.kaggle.com/datasets/shivamb/netflix-shows
- Content: Movies and TV Shows available on Netflix  
- Key features: Contains information about 8,800+ movies and TV shows, including title, director, cast, country, release year, rating, and description.


---
